Changelog

Version 2.3 - Logging Update

February 21, 2025

  • Channel Creation Logs – ZenBot now logs when a new channel is created, including the creator and timestamp.
  • Channel Deletion Logs – Track when a channel is deleted, along with who deleted it and when.
  • Channel Permission Update Logs – Get notified when channel permissions change, including the before-and-after states.

Version 2.2 - Fixed Bugs

February 15, 2025

  • Fixed $giverole Exploit – Now more stable and secure.
  • Improved Warn System – When issuing a warn, it will now be logged in member logs automatically.
  • Fixed Typographical Errors – Enhanced text clarity for a smoother experience.

Version 2.1 - New Commands & Website

February 8, 2025

  • Revamped $help Command – Now more intuitive and easier to use.
  • New Website – Explore ZenBot’s features and updates in one place.
  • $memberlogs Command – Set a dedicated channel for member logs.
  • Global Points System – Points now work across all servers ZenBot is in!
